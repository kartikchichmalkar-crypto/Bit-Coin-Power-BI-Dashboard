# 📊 Bitcoin Dashboard – Power BI
📌 Overview

This Power BI dashboard provides interactive visual insights into Bitcoin market trends, including price movements, trading volume, and key performance indicators. It is designed to help users analyze historical data and monitor cryptocurrency behavior effectively.
